## 二、综合问题总结

### 🔴 严重问题

| 问题 | 影响 | 涉及题目 |
|------|------|----------|
| **3选项题目过多** | 5道题只有3个选项，打破4选1的一致性 | Q14, Q16, Q17, Q18, Q19 |
| **Q8权重爆炸** | 一道题影响4个维度，结果高度依赖此题 | Q8 |
| **Q19维度单一** | 全部3个选项只测E，浪费题目位置 | Q19 |
| **W维度只有正向** | 13个选项给W加分，0个选项扣W分 | Q1,2,5,7,8,10,11,13,14 |
| **D维度采样不足** | 全卷只有Q1和Q3涉及D | - |

### 🟡 中等问题

| 问题 | 说明 |
|------|------|
| **维度映射逻辑混乱** | 很多选项的维度归属是"硬凑"的 |
| **E维度题目过多** | 几乎每道题都能影响E，导致E分数普遍偏高 |
| **A维度两极分化** | 要么+2（卷王）要么-2（摆烂），缺乏中间状态 |
