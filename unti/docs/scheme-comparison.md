## 用户建议：4维对立轴方案

### 新维度设计

```
轴1: 行动(Doer) ←→ 内耗(Thinker)
轴2: 外向(Social) ←→ 独处(Solo)  
轴3: 稳定(Stable) ←→ 发疯(Chaos)
轴4: 卷王(Grind) ←→ 躺平(Rest)
```

### 优势

| 优点 | 说明 |
|------|------|
| **逻辑更清晰** | 每个维度有明确对立面，不是正负值 |
| **映射更直接** | 人格 = 最高维度直接决定 |
| **避免5维混乱** | D和A重叠问题消失 |
| **用户好理解** | 结果解释更简单 |

### 人格映射（4维 → 9人格）

```
高Doer + 高Stable    → 👑 KING（六边形战士）
高Doer + 高Chaos     → 🌪️ WILD（发疯但能干）

高Thinker + 高Solo   → 🪨 NULL（隐形NPC）
高Thinker + 高Social → 🎭 OOPS（倒霉蛋）

高Social + 高Stable  → 🌸 PURE（清澈者）
高Solo + 高Stable    → 🪨 NULL（也是）

高Grind + 高Chaos    → 🔥 FIRE（赶Due狂徒）
高Rest + 高Thinker   → 🛌 FLOP（仰卧者）

特殊组合：
高Doer + 高Rest + 低Thinker → 🌌 FREE（旷野）SSR
高Doer + 高Solo + 高Chaos   → 👁️ ALIEN（局外人）UR
```
